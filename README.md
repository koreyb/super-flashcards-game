An adaptive flashcard game for kids ages 3, 6, and 8 with progressive learning AI.

## Features
- **Age 3:** Letters, Numbers, Colors, Shapes
- **Age 6:** Basic Math (Addition, Subtraction, Multiplication, Division)
- **Age 8:** Advanced Math with mixed operations
- **Adaptive Learning:** Focuses on areas that need improvement
- **Gamification:** Scoring, streaks, levels, celebrations
